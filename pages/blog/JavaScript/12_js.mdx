## 배열 분해하기

```jsx
let arr = ["Bora", "Lee"]
let [name1, name2] = arr;
alert(name1) // Bora
alert(name2) // Lee
```

```jsx
//쉼표를 활용하면 특정 요소는 무시할 수 있다.
let arr = ["bora", "lee", "park", "sungmin"]
let [name1, name2, , name3] = arr;
alert(name1) // bora
alert(name2) // lee
alert(name3) // sungmin

//할당 연산자 우측에는 모든 이터러블(반복 가능객체)가 올 수 있다.
let [a, b, c] = "abc"; // ["a", "b", "c"]

//할당 연산자 좌측에는 모든게 올 수 있다.
let user = {};
[user.name, user,surname] = "Bora Lee".split(" ");

//...를 활용해 나머지 요소를 가져 올 수 있다.
let [name1, name2, ...rest] = ["Julius", "Caesar", "Consul", "...names"];
//rest[0] // Consul
//rest[1] // ...names
//rest.length //2

//변수간의 바꿔치기도 간단하게 진행할 수 있다
let a = 1;
let b = 2;
[a, b] = [b, a];
```

## 객체 분해하기

객체도 동일한 방식의 문법으로 분해할 수 있다.

```jsx
let options = {
	title: "Menu",
	width: 100,
	height: 200,
};

let {title, width, height} = options;

alert(title) // "Menu"
alert(width) //  100
alert(height) // 200

//각각의 목표를 지정하고, 원하는 변수로 지정할 수도 있다.
let {title: id, width: w, height: h} = options;
alert(id) // "Menu"
alert(w) // 100
alert(h) // 200

//이러한 목표지정은 함수호출을 사용할 수도 있다, 물론 key가 할당되어 있지 않을 때만, 이를 동시에도 가능
let {wid = prompt("width?"), tit = 100, title)} = options;
alert(wid) // prompt 입력 값
alert(tit) // 100
alert(title) // "Menu"

//Array와 동일하게 rest parameter를 활용해 나머지를 저장할 수도 잇다.
let {title, ...rest} = options;
alert(rest.height) // 200
alert(rest.width) // 100
```

## 중첩 분해하기

Array와 객체를 다양하게 사용하는 구조의 경우에도, 구조분해 할당을 적절히 활용하면 분해 할 수 있다.

```jsx
let options = {
    size: {
        width: 100,
        height: 200,
    },
    items: ['Cake', 'Donut'],
    extra: true,
};
// 코드를 여러줄에 걸쳐 작성해서 의도하는 바를 나타내자.
let {
    size: { width, height },
    items: [...item],
    extra: bool,
    title = "new Title",
} = options;

alert(width) // 100
alert(height) // 200
alert(item[0]) // "Cake"
alert(item[1]) // "Donut"
alert(bool) // true
alert(title) // "new Title"
```

함수에게 전달할 매개변수로서도 이는 활용될 수 있다.

```jsx
let options = {
  title: "My menu",
  items: ["Item1", "Item2"]
};

const showMenu = ({title = "Untitled", width = 200, height = 100, items = []}) => {
	alert(`${title} ${width} ${height}`); // My Menu 200 100
	alert( items ); // Item1, Item2
}
```
