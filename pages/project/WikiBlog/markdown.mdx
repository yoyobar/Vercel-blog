# MDX

확장된 문법의 사용법을 다뤄보겠습니다.

현재 블로그에서는 `tailwind`를 따로 추가했으므로, 사용이 가능합니다.

export function Counter({ children }) {
    const [count, setCount] = useState(0);
    return (
        <>
            <div className='w-full flex justify-center'>
                <button
                    className='w-[128px] font-bold p-2 rounded-md bg-indigo-400 text-white hover:bg-indigo-600'
                    onClick={() => setCount(count + 1)}
                >
                    {children}
                    {count}
                </button>
            </div>
        </>
    );
}

<Counter>컴포넌트 클릭 </Counter>

`{children}`는 컴포넌트 안에 작성되는 내용을 `Props`로 전달받고 그 외는 일반적인 리액트와 동일합니다.

## Github Custom `Markdown`

깃허브에서 따로 추가된 커스텀 `Markdown` 문법을 지원합니다.

### 취소선
~~삭제됨~~

```md filename="Markdown" copy
~~삭제됨~~
```

### 투두 리스트
<br/>
-   [x] 위키 내용 작성
-   [ ] 프로젝트 내용 추가
-   [x] 투두리스트 다뤄보기

```md filename="Markdown" copy
- [x] 위키 내용 작성
- [ ] 프로젝트 내용 추가
- [x] 투두리스트 다뤄보기
```
