# 탭

탭은 한정적인 내용을 구별하여 담을 수 있는 `Nextra` 내장 기능입니다.

```mdx
import { Tabs } from 'nextra/components'
```
를 상단에 선언하여 사용 할 수 있습니다.


<Tabs items={['pnpm', 'npm', 'yarn']}>

  <Tabs.Tab>2. 자바스크립트 언어의 패키지 관리자 (Node.js)</Tabs.Tab>
  <Tabs.Tab>3. 소프트웨어 패키징 시스템 (Facebook)</Tabs.Tab>
</Tabs>
