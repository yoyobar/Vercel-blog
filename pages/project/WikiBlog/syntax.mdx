### 라인 강조
코드 블럭내에서 `{1, 4-5}`안에 특정 번호를 넣을 경우, 강조 표시가 가능합니다.
```js {1, 4-5}
import {useState} from 'react'

function Counter() {
    const [count, setCount] = useState(0)
    return <button onClick={() => setCount(count + 1)}>{count}</button>
}
```

### 문자열 강조
선언부에 `/{useState}/` 를 활용하여 문자를 강조 표시가 가능합니다.
```js /useState/
import {useState} from 'react'

function Counter() {
    const [count, setCount] = useState(0)
    return <button onClick={() => setCount(count + 1)}>{count}</button>
}
```

### 코드 번호 표시
선언부에 `showLineNumbers`를 활용하여 넘버 표시가 가능합니다.
```js showLineNumbers
import {useState} from 'react'

function Counter() {
    const [count, setCount] = useState(0)
    return <button onClick={() => setCount(count + 1)}>{count}</button>
}
```

### 파일 이름 표시
선언부에 `filename`를 활용하여 이름 표시가 가능합니다.
```js filename="파일 이름 표시 예시"
import {useState} from 'react'
```
